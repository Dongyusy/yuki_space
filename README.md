# Yuki Portfolio

Astro-based personal portfolio site for local editing, Git version control, and future GitHub Pages deployment.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content

- Main page: `src/pages/index.astro`
- Profile and contact data: `src/data/profile.ts`
- Experience, projects, and campus practice data: `src/data/portfolio.ts`
- Images: `public/images/`

Replace placeholder copy in the data files first. The image paths are already normalized for web deployment.
